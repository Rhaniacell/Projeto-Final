Exemplo de Código HTML


        Exemplo de uma estrutura básica HTML: 
        <!DOCTYPE html>
        <html lang="pt-BR">
        <head>
            <meta charset="UTF-8">
            <title>Minha Página</title>
        </head>
        <body>
            <h1>Bem-vindo à minha página!</h1>
            <p>Este é um exemplo simples de HTML.</p>
        </body>
        </html>

Exemplo de Estilo CSS


        Estilo básico para título:
        h1 {
            color: #333;
            font-family: Arial, sans-serif;
            text-align: center;
        }

        Estilo para parágrafo:
        p {
            color: #555;
            line-height: 1.6;
        }

Dicas para Iniciantes

  • Utilize tags semânticas como <header>, <footer> e <section> para organizar o conteúdo.
  • Centralize e organize seu CSS para uma manutenção mais fácil.
  • Use classes e IDs para aplicar estilos específicos e reutilizáveis.
